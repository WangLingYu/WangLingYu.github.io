C++指针问题

Created by Coder on 2016/11/15.

C++各种不同类型函数调用以及相应操作效果

本文用代码来打印出对象内存地址来实际解释说明C++中各种函数调用的具体对象内存发生情况，作者能力有限，如有错误，希望读者能指出错误

#include using namespace std;
class Test {
public:
string name;
};

首先声明，下文中的函数1，3，4在调用fun(test)情况下无法共存，读者自己取消注释查看结果即可

######该函数编号为函数1，无法和fun(Test &test)共存，因为main函数中调用fun(test)无法识别改调用该函数还是fun(Test &test)

//void fun(Test test) {
// cout << “fun(Test test)函数中test的值” << &test << endl;//通过打印得到test的值并不是外面test的地址，说明此时改变test不会同步改变外部test的值
//解释 函数会在该函数的作用域内临时生成一个对象，将外界test函数的值全部复制给自己，在fun函数结束后自身被析构掉,被销毁，但是外界的test对象地址值没有收到改变，因此值不变
// （自己暂时留坑，今后再验证，考虑如果对象里面有指针成员，会不会临时变量和外界对象会不会指向同一个地方）
test.name = “test函数”;
}

######该函数编号为函数2，当main函数中调用的是fun(p),或者fun(&test)时会调用该函数
void fun(Test test) {
cout << “fun(Test test)函数中test的值” << test << endl;//通过打印得到test的值也就是外面test的地址，说明此时改变test也在同步改变外部test的值
test->name = “test函数”;
}

######该函数编号为函数3，我尝试重写fun函数只是参数不同，但是调用时每次都调用的都是fun(Test *test),可能是main函数传入的指针匹配不了fun(Test 7test)函数,除非main函数中调用时传入的是test,此时也是传入main函数中test的自身地址，所以修改test的值也会同步修改外界test的值

//void fun(Test &test) {
// cout << “fun(Test &test)函数中test的值” << &test << endl;//通过打印得到test也是外部test的地址,此时 test不是指针，直接修改test就会同步修改外部a的值
// test.name = “&test函数”;
//}

######该函数编号为函数4，测试发现&fun(Test &test)结果也是相同的，只不过如果参数是&test，那么传入的test的值也会同步变化
//Test &fun(Test test) {
// cout << “&fun(Test test)函数中test的值” << &test << endl;
//通过打印得到test的地址,test是一个临时变量
// test.name = “&fun(Test test)”;
// return test;
//}

######返回一个指针，指针的值为test的地址
Test fun(Test test) {
cout << “fun(Test test)函数中test的值” << &test << endl;//通过打印得到test的地址,test是一个临时变量
test.name = “*fun(Test test)”;
return &test;
}

int main() {
Test p;
Test test;
p = &test;
cout << “p的值为(也就是test的地址值):” << p << endl << endl;
fun(p);//此时是把p地址上的内容，也就是a的地址传入
cout << “调用fun(Test test)后test.name的值为：” << p->name << endl << endl;
fun(&test);//传入test自身的地址，实际上就是p的内容，和fun(p)效果一样
cout << “调用fun(Test test)后test.name的值为：” << p->name << endl << endl;// fun(test);
// cout << “调用fun(Test &test)后test.name的值为：” << p->name << endl << endl;
// fun(test);
// cout << “调用fun(Test test)后test.name的值为：” << p->name << endl << endl;
// Test test1 = fun(test);
// cout << “调用&fun(Test test)后test1的地址值为：” << &test1 << endl;
// cout << “调用&fun(Test test)后test1.name的值为：” << test1.name << endl << endl;
Test test1 = fun(test);
//用于调用fun(Test test);
cout << “调用fun(Test test)后test1的地址值为：” << &test1 << endl;
cout << “调用*fun(Test test)后test1.name的值为：” << test1.name << endl << endl;
return 0;
}
