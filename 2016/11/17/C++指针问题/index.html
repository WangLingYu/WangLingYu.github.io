说明，此代码由浅入深，同时可能解说思想有所不同，让人觉得怪异，但是个人认为这种思想有助于理解C，C++实质性的东西。下面基本涵盖了C语言和C++的简单指针部分，同时也简单介绍了C++特有的引用。

#include <iostream>

using namespace std;

void isChangePtr(int *n);   //测试改变指针本身的地址

int main() {

int *p;
int *q;

这里p,q并不是指取得p地址上储存的值,而是另一种用法，用于声明p,q是一个指针，实际上
是与int放在一起的， 是int p , int* q;

int a = 6;
p = &a;
cout << "    "<<&*p<<endl;
cout << "a自身的地址为:" << &a << endl;
cout << "a的值,也就是a的地址所储存的值为:" << a <<  endl;
cout << "p自身的地址为:" << &p << endl;
cout << "p的值，也就是p自身地址所储存的值（也就是a自身的地址）为:" << p << endl;
cout << "*p,也就是p自身地址所储存的值（也就是a的自身地址）所储存的值为:" << *p<< endl;

总结：a指的是a地址所储存的值，&a是取a自身的地址，p指的是p地址所储存的值，指针的原理就是开辟一个新空间来储存目标的地址，所以&p是p自身的地址，p代表就是a的地址，*p指的是取p地址上储存的值（也就是a的地址）所储存的值

q = p;
cout << "q自身的地址为:" << &q << endl;
cout << "q自身地址上储存的值（也就是p自身地址所储存的值，这里就是a的地址）为:" << q<< endl;
cout << "*q,也就是q自身地址所储存的值（也就是a的地址）所储存的值为:" << *q << "\n\n";

*p = 7;
cout <<"令a = 7,当a的值改变过后"<<endl;
cout << "*p,也就是p自身地址所储存的值（也就是a的自身地址）所储存的值为:" << *p << endl;
cout << "*q,也就是q自身地址所储存的值（也就是a的地址）所储存的值为:" << *q << endl;
cout << "a的值,也就是a自身地址所储存的值为:"  << a << "\n\n\n";

q =p 的实质就是将p地址上储存的值（也就是a的地址）赋予给q，这样q的地址所储存的值就是a的地址,所以此时q也指向a，所以此时改变a的值，那么q和p所代表的值都会变化

cout << "测试在其他函数中对指针进行处理后的变化"<<endl;
isChangePtr(p);
cout << "p自身的地址为:" << &p << "\n";
cout << "p的值，也就是p自身地址所储存的值（也就是a自身的地址）为:" << p << "\n";
cout << "*p,也就是p自身地址所储存的值（也就是a的自身地址）所储存的值为:"<< *p << "\n\n\n";

总结：在C语言中若函数形参是void isChangePtr(int n) ,也就是n = p;

###这只是把p地址储存的值（也就是a的地址赋予给n），那么函数中对n进行操作，也相当于对a进行操作，例如如果在函数中把a的值改为8，那么在函数外*p的值也会改变为8,但是这样并没有对p进行任何操作，那么假设若在函数中让n不指向a，让n指向b，但是对函数外的p指针没有任何影响，p指针还是指向a

##//测试指向指针的指针
cout << “测试指向指针的指针” << endl;
int n;
int b = 5;
n = &b;
int **m;
m = &n;
cout << “b的值为：” << b<<endl;
cout << “&b的值为，也就是b自身地址为：” << &b<<endl;
cout << “&n的值为，也就是n自身地址为：” << &n <<endl;
cout <<”n的值为，也就是n自身地址上储存的值（也就是b的地址）的地址为：” << n << endl;
cout <<”n的值为，也就是n自身地址上储存的值（也就是b的地址）所储存的值为：” << n << endl;
cout <<”&m的值为，也就是m自身的地址为：” << &m << endl;
cout <<”m的值为，也就是m自身地址上储存的地址（也就是n的地址）为：” << m << endl;
cout <<”m的值为,也就是m自身地址上储存的地址（也就是n的地址）上所储存的值（也就是b的地址）为：” << m << endl;
cout <<”m的值为,也就是m自身地址上储存的地址（也就是n的地址）上所储存的值（也就是b的地址）上所储存的值（也就是b代表的值）为” << m << endl;
cout << “&m的值为：（也就是n的地址）”<< *&m<<endl;
cout << endl << endl;

###总结：所谓的指向指针的指针，也就是开辟新的区域来存放之前的指针，假设m是第二个指针，它指向第一个指针n,n指向一个int类型的值a。m代表的是m地址(m自身地址)上储存的地址(m储存的值，也就是n指针自身的地址)，m代表的是取m地址(m自身地址)上存放的地址（m储存的值,也就是n指针自身的地址）上所储存的地址(也就是n指针地址储存的值,也就是a的地址，注意，是a的地址，而不是a的值)，*m代表的是取m地址(m自身地址)上存放的地址（m储存的值,也就是n指针自身的地址）上所储存的地址(也就是n指针地址储存的值,就是a的地址，注意，是a的地址，而不是a的值)所储存的值（此时就是5）。希望读者能够自己自行进行理解，推荐用0，1，2，3，4的数字来代表各个变量的地址，进行手动画图的方式进行理解

##//测试C++特有的引用
int e = 5;
int &x = e;
int &y = x;
cout << “e的值为，也就是e自身地址上所储存的值为：”<<e<<endl;
cout << “&e的值为，也就是e自身地址为：”<<&e<<endl;
cout << “x的值为,也就是x自身地址所储存的值为：”<<x<<endl;
cout << “&x的值为，也就是x自身的地址为：”<<&x<<endl;
cout << “y的值为,也就是y自身地址所储存的值为：”<<y<<endl;
cout << “&y的值为，也就是y自身地址为：”<<&y<<endl;

###总结：C++中特有的引用，也就是&x = e;这种用法，不用重新开辟一块区域来专门存放x，而是x的地址就是e的地址,真正意义上的别名，改变x,也会跟着改变e的值
